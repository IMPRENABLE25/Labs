# Labs
L’ensemble des projets présentés dans ce dépôt a été réalisé en environnement virtualisé, à l’aide d’outils professionnels tels que GNS3, VirtualBox et VMware. Ces plateformes ont permis de simuler des infrastructures réseau et systèmes complètes, offrant un cadre idéal pour pratiquer, tester et documenter chaque lab dans des conditions proches d’un environnement réel.
